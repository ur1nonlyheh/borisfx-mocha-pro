# Boris FX Mocha Pro 2024

This repository contains **Boris FX Mocha Pro 2024**, provided as a single archive with the **standalone application** and **plugin versions** for multiple host platforms.

## Included Versions

- **Mocha Pro Standalone**
- **Mocha Pro for Adobe** (After Effects, Premiere Pro)
- **Mocha Pro for Avid** (Media Composer)
- **Mocha Pro for OFX** (Nuke, Fusion, DaVinci Resolve, HitFilm, VEGAS Pro)

## Overview

Mocha Pro is a professional planar tracking and rotoscoping tool used for object removal, masking, stabilization, and camera tracking in post-production workflows.

## Key Features

- Award-winning planar tracking
- Advanced roto and masking tools
- 3D Camera Solver powered by SynthEyes
- Object removal and stabilization
- GPU-accelerated performance

## Installation

1. Download and extract the archive.
2. Choose the required version (Standalone or Plugin).
3. Run the installer and follow the on-screen instructions.
4. Launch the application or host and verify the plugin is available.


## Notes

Feature availability may vary depending on the host application.
